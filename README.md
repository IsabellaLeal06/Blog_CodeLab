# 💻 CodeLab

O CodeLab é um blog responsivo desenvolvido com HTML e CSS, criado com foco em design moderno, experiência do usuário e organização de conteúdos sobre tecnologia e programação.

O projeto apresenta uma interface limpa e intuitiva para leitura de artigos, utilizando conceitos de responsividade, estilização moderna e estrutura semântica.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts
- - JavaScript

---

## 🎨 Funcionalidades

- Layout moderno
- Barra de pesquisa
- Cards de postagens
- Interface responsiva
- Design dark mode
- Estrutura organizada e semântica

---

## 📱 Responsividade

O projeto foi desenvolvido para funcionar em:

- Desktop
- Tablets
- Smartphones
